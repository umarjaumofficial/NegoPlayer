================================================================================
                          TROUBLESHOOTING
================================================================================

Problem: "Keystore not found"
────────────────────────────
Solution: bash setup-keystore.sh

Problem: "Build failed"
──────────────────────
Solution: ./gradlew clean build --stacktrace

Problem: "Cannot push to GitHub"
────────────────────────────────
Solution: 
  1. Verify remote: git remote -v
  2. Update if needed: git remote set-url origin https://github.com/Umarjaum/NegoPlayer.git
  3. Try again: git push origin main

Problem: "APK too large"
────────────────────────
Expected: 40-45 MB is normal (optimized)
Use Play Store AAB for device-specific variants

Problem: "Cannot install APK on device"
──────────────────────────────────────
Solutions:
  1. Enable "Unknown Sources" in device settings
  2. Check API level compatibility (min: 21, target: 34)
  3. Try: adb uninstall com.negoplayer (remove old version)
  4. Try: adb install -r app-release.apk (reinstall)

More Help: See COMPLETE_SETUP_GUIDE.md → Troubleshooting
